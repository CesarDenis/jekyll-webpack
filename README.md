## Development

* Cloning the repo

```bash
git clone https://github.com/CesarDenis/jekyll-webpack.git
```

* Installing dependencies

```bash
npm install
```

* Running scripts

| Action                    | Usage          |
| ------------------------- | -------------- |
| Starting development mode | `npm start`    |
| Linting code              | `npm run lint` |
